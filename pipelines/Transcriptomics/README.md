# 🧬 Transcriptomics

## Overview

This section contains reproducible pipelines for transcriptomic data analysis using bulk RNA-seq and related technologies.

The projects cover the complete workflow from raw sequencing reads to biological interpretation.

---

## Learning Objectives

After completing these projects you will be able to:

- Perform RNA-seq quality control
- Align sequencing reads
- Quantify gene expression
- Identify differentially expressed genes
- Perform pathway enrichment analyses
- Build co-expression networks
- Analyze coding and non-coding RNAs
